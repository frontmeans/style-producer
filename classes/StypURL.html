priority: number

Inherited from StypZero.priority

  • Defined in style-producer/src/value/value.ts:44

CSS property value priority.

The value StypPriority.Important and above means the property is !important. Everything else means normal priority.

The property value with higher priority number takes precedence over the one with lower one.

  • by(source: StypValue): StypURL | undefined
    • Defined in style-producer/src/value/url.ts:39

    Maps the given CSS property value to URL. Defaults to undefined if mapping is not possible.

    This method allows to use a StypURL class as [CSS property mapping]StypMapper.Mapping.

    Strings are treated as URLs (without url() functional syntax). !important suffix is respected.
